🌸 MoodCompanion – Your Gentle Digital Counselor

MoodCompanion is a calming, user-friendly web application designed to support mental well-being, especially for IT professionals and students. It provides simple, interactive tools to help users relax, reflect, and improve their mood through guided activities.

✨ Features

😊 Mood Selection

* Users can select their current mood:

  * Distressed 😫
  * Happy 😊
  * Neutral 😐
  * Sad 😔
  * Tired 😴
  * Angry 😡


🎵 Music Therapy

* Choose:

  * Music type (Nature, Relaxing, Calm)
  * Language (Sinhala, English, Tamil, Hindi, Korean)
* Get **10 YouTube links** instantly

🎬 Movie Suggestions

* Select genre (Funny, Action, Sad, Sci-Fi, etc.)
* Choose language
* Get **10 trailer links**
